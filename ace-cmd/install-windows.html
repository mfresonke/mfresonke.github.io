OLD Windows Install Instructions

Download Cygwin

Most of you should download the 64-bit Installer.
However, some of you are sticking it out on 32-bit Windows, so you'll need the 32-bit Installer.

Run Installer

Should be as easy as it sounds. When asked for the mirror, do the scientific thing and blindly choose one. (That's what I do.)

Install Some Additional Packages

When asked for additional packages, use the search feature to install and find the following:

  • gcc-core (Devel)
  • gcc-g++ (Devel)
  • make (Devel)
  • git (Devel)
  • wget (Web)
  • fortune-mod (Games)
  • perl (Interpreters)
  • nano (Editors)
  • figlet (Text)
  • espeak (Audio)
You only need to click the "Bin" part for it to install.
If none of this makes sense (or even if it does), don't worry. I'll demonstrate on the board how to install it.

Download Cowsay

Unfortunutely, cygwin does not come with an absolutely essential program: cowsay. (I kid, I kid.) Click here to download it.

Install Cowsay

Not gonna lie, this is the messyest part of the windows install process. Even so, I just couldn't give up the awesome learning power of the cow.
To install this beast, extract the .zip and drag the "usr" folder into your cygwin directory. This will be installed at your root (C:) drive. Accept any overwrite warnings, it's NBD.

Link Windows Home to Cygwin Home (Technically optional, but you should really do it)

Open an "Administrative" Command Line Prompt. (Open start menu, type "cmd", right-click "Run As Administrator")

64-bit Command to Paste

mklink /D C:\cygwin64\home\%USERNAME%\win-home %HOMEDRIVE%%HOMEPATH%

32-bit Command to Paste

mklink /D C:\cygwin\home\%USERNAME%\win-home %HOMEDRIVE%%HOMEPATH%
