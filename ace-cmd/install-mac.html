Mac Install Instructions

Install Homebrew (If You Haven't Already)

Either go to the website and view their instructions there, or open a terminal (search Spotlight for "Terminal") and paste the following:
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
Follow any onscreen instructions.

Install Some Packages

Now let's use brew to install some stuff!
brew update && brew install fortune cowsay git make wget

THAT'S IT! :D
