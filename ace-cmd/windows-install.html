Windows Install Instructions

Download Cygwin

Most of you should download the 64-bit Installer.
However, some of you are sticking it out on 32-bit Windows, so you'll need the 32-bit Installer.

Install Some Packages

In addition to the base packages, you'll want to install the following:

  • gcc-core
  • gcc-g++
  • make
  • gdb
  • git
  • wget
  • fortune-mod
  • perl
  • openssh
You can use the search field to find these efficiently. I will demonstrate on the board how to install them.
Not all of these will be used today, but they are all useful.
